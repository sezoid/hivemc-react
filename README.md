# hivemc-react
Simple web-application for viewing statistics on server The Hive

### How to build
```
$ git clone https://github.com/sezoid/hivemc-react.git
$ cd hivemc-react
$ yarn install
$ yarn build
```

### How to run
```
$ cd hivemc-react
$ yarn start
```

Check your bundles in a 'build' folder.
